GreenCloud

Carbon-aware self-hosted PaaS

GreenCloud is a personal project building a carbon-aware Platform-as-a-Service running on a Raspberry Pi 5. It's currently in active development — the goal is a fully self-hosted deployment pipeline that tracks its own carbon footprint and defers non-critical work to times when the electricity grid is cleanest. Everything below is live and running from home with zero open ports, powered by Cloudflare Tunnel.
